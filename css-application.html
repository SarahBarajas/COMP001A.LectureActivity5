External CSS

An external style sheet can be written in any text editor, and must be saved with a .css extension. The external .css file should not contain any HTML tags.

Internal CSS

An internal CSS is used to define a style for a single HTML page. An internal CSS is defined in the head section of an HTML page, within a style element.

Inline CSS

The style attribute specifies an inline style for an element. The style attribute will override any style set globally,
